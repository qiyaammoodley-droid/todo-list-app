# To-Do List App

This project is a simple web-based to-do list application that allows users to manage daily tasks.

Users can add tasks, mark them as completed, and delete them when finished.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks from the list
- Simple and clean user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Purpose

This project was built as part of a beginner developer portfolio to demonstrate basic JavaScript functionality and DOM manipulation.

## How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
